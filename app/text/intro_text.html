CAIC-RT: COVID-19 Acute and Intensive Care Resource Tool

Authors

  • Vasily Giannakeas, MPH

  • Matthew T. Warkentin, MSc
  • Isaac I. Bogoch, MD

Description

An online tool capable of estimating the maximum daily number of COVID-19 cases your healthcare system could sustain based on age-based case distribution and severity and the number of appropriable acute and critical care resources.

How to use this tool:

  1. First, review the default age-based case distribution and severity which is based on US data from the CDC.
    • Optional: You may enter your own custom table based on local clinical data and experience.
  2. Second, tune the Expected Length of Stay and Resource Availability for COVID-19 Cases parameters to your own environment.
  3. Third, inspect the bar graph which presents the maximum daily number of COVID-19 cases your healthcare system could sustain based on acute care beds, critical care beds, and mechanical ventilators.
  4. Fourth, visualize how the results will change when modifying input parameters.
  5. Fifth, generate a report to keep a copy of the results!
