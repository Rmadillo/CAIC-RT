Description:

An online tool capable of estimating the maximum daily number of incident COVID-19 cases that a healthcare system could manage based on age-based case distribution and severity, and the number of appropriable acute and critical care resources.

How to use this tool:
  1. Review the default age-based case distribution and severity, which is based on US data from the CDC.
    • Optional: You may enter your own information into the table based on local clinical data and experience.
  2. Tune the inputs for Expected Length of Stay for COVID-19 Cases and Resource Availability for COVID-19 Cases to your own healthcare environment.
  3. Inspect the results in the bar graph, which presents the maximum daily number of incident COVID-19 cases that a healthcare system could sustain based on acute care beds, critical care beds, and mechanical ventilators.
  4. Visualize how the results change when modifying the input parameters.
